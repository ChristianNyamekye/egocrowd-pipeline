**Step 3: Implement the two-stage pipeline inside the function.**

Stage 1 -- GroundingDINO hand detection:
- Load GroundingDINO model (same as v5_hand_detect.py)
- Detect hands in each frame with text prompt "hand"
- Extract bounding boxes, filter by confidence threshold 0.2

Stage 2 -- HaMeR mesh recovery (attempt):
- Try to load HaMeR model
- For frames with detected hands, crop to bounding box, resize to 256x192
- Run HaMeR inference to get MANO params and 3D joints
- Extract wrist position (joint 0) as `wrist_3d_camera`
- Estimate grasping from thumb-index distance (joints 4 and 8, threshold 0.04m)

If HaMeR loading fails (dependency issues), fall back to GroundingDINO-only mode:
- Use bounding box center as wrist_pixel
- Set wrist_3d_camera to None
- Estimate grasping from hand detection confidence + box size heuristic

```python
def extract_hand_poses(
    rgb_dir: str,
    hand_boxes: Optional[dict] = None,
    device: Optional[str] = None,
) -> dict:
    """Extract 3D hand meshes from RGB frames using HaMeR on Modal GPU.

    Calls the Modal function defined in processing/hamer_modal.py.
    Falls back gracefully if Modal is unavailable.

    Args:
        rgb_dir: Directory containing RGB frames (.jpg)
        hand_boxes: Optional pre-computed hand bounding boxes (unused, HaMeR detects internally)
        device: Ignored (runs on Modal GPU)

    Returns:
        dict with per-frame MANO parameters, joint positions, and translations.
        Format: {"timesteps": [...], "detection_rate": float, "model": "hamer"}
    """
    from pathlib import Path

    rgb_path = Path(rgb_dir)
    frames = sorted(rgb_path.glob("*.jpg"))
    if not frames:
        raise FileNotFoundError(f"No .jpg frames found in {rgb_dir}")

    print(f"Processing {len(frames)} frames with HaMeR (Modal GPU)...")

    # Load frame bytes
    frame_bytes = []
    for f in frames:
        frame_bytes.append(f.read_bytes())

    # Call Modal function
    import modal
    run_hamer = modal.Function.lookup("flexa-hamer", "run_hamer_inference")
    result = run_hamer.remote(frame_bytes)

    detection_rate = result.get("detection_rate", 0)
    n_detected = sum(1 for ts in result.get("timesteps", []) if ts.get("detected"))
    print(f"HaMeR detection: {n_detected}/{len(frames)} frames ({detection_rate:.1%})")

    return result
```

```python
def cam_to_world(point_cam, pose):
    """Transform a camera-frame point to world frame using R3D pose."""
    tx, ty, tz = pose[0], pose[1], pose[2]
    qw, qx, qy, qz = pose[3], pose[4], pose[5], pose[6]
    R = np.array([
        [1-2*(qy**2+qz**2), 2*(qx*qy-qw*qz), 2*(qx*qz+qw*qy)],
        [2*(qx*qy+qw*qz), 1-2*(qx**2+qz**2), 2*(qy*qz-qw*qx)],
        [2*(qx*qz-qw*qy), 2*(qy*qz+qw*qx), 1-2*(qx**2+qy**2)]
    ])
    return R @ np.array(point_cam) + np.array([tx, ty, tz])
```
